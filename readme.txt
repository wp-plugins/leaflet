Plugin to embed maps (often OpenStreetMap) with personal markers in your blog.

== Description ==

Licenses:

* OpenStreetMap: [OpenStreetMap License](http://wiki.openstreetmap.org/wiki/OpenStreetMap_License) 
*  Leaflet by Cloudmade, 2011 (http://leaflet.cloudmade.com)

== Installation ==

1. Upload OSM folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create your map in Leaflet -> Maps or simply insert shortcode with mlat,mlon,[mtext] and [micon] parameters.

== Frequently Asked Questions ==

= Do I need any key or any registration to show maps in my blog =

No!
